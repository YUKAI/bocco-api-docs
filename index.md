## BOCCO API とは

BOCCO を Restful API を使って操作する方法を解説したドキュメントです。BOCCO に自由な言葉をしゃべらせたりデータを取得したりすることができます。

BOCCO API の現在のステートはベータであり、実験的な段階にあります。
